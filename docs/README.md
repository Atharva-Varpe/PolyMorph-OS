## 📖 About PolyMorph

PolyMorph is a flexible Linux distribution installer that gives users maximum choice in configuring their system:

- **Multiple base distributions** (Arch, Debian, Ubuntu, Fedora, etc.)
- **Choice of desktop environments** (KDE, GNOME, XFCE, i3, etc.)
- **Init systems** (systemd, OpenRC, runit, s6)
- **Kernels** (LTS, Zen, Hardened, custom)
- **Filesystems** (ext4, Btrfs, XFS, F2FS, ZFS)
- **And much more...**
